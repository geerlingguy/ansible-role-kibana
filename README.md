# Ansible Role: Kibana

An Ansible Role that installs Kibana on RedHat/CentOS or Debian/Ubuntu.

## Requirements

None.

## Role Variables

For variables please see the [kibana documentation](https://www.elastic.co/guide/en/kibana/current/settings.html) 


## Dependencies

None.

## Example Playbook

    - hosts: kibana
      roles:
        - stuvusIT.kibana
